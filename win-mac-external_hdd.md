# How to install Windows on external drive on Mac

Notes for installing Windows on external drive on Macbook. Goal is to leave original, internal disk untouched, that is why Bootcamp is not used.

## Advantages
* original MacOS installation is untouched

## Disadvantages
* on boot must be selected disk with Windows to boot from
* from Windows is not working "reboot to MacOS"


## What is needed
* Macbook
* external drive, preferrably USB3 SSD (because of speed)
